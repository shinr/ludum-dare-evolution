the world tree is a game created for ludum dare 24 (www.ludumdare.com), a 48 hour game making compo.

your mission is to evolve the world tree by collecting the artifacts in ever evolving terrain.

arrow keys move you left, right and up and down key is used to pick stuff up
clicking stuff with mouse cursor is helpful

created with python 2.7+pyglet 1.1.4
sounds with Massive, native instruments
additional sounds with sfxr, thanks DrPetter!
